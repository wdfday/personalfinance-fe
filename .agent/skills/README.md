# Frontend Skills

This directory will contain skills from vercel-labs/agent-skills.

## Installation

Run the following command to add Vercel's agent skills:

```bash
cd client
npx add-skill vercel-labs/agent-skills
```

This will install community-maintained skills for:
- Next.js best practices
- React patterns
- TypeScript conventions
- Vercel deployment
- UI/UX design patterns

## Available Skills

After installation, you'll have access to skills like:
- `nextjs-app-router` - App Router patterns and best practices
- `react-server-components` - RSC usage guidelines
- `tailwind-design-system` - Tailwind CSS patterns
- `vercel-deployment` - Deployment strategies
- And more...

## Usage

When working in the `client/` directory, Antigravity will automatically detect and use these skills for context-aware assistance.
